# WaveViT 网络结构流转图

## 一、Transformer Block 结构流转图（Wave 类型）

```
输入 x (B, N, C)
│
├──────────────────────────────────────────────────────────┐ (residual)
│                                                          │
▼                                                          │
LayerNorm                                                  │
│                                                          │
▼                                                          │
┌─────────────── WaveAttention ──────────────────────┐     │
│                                                    │     │
│  输入 x (B, N, C)                                  │     │
│  │                                                 │     │
│  ├──────────────────┐                              │     │
│  │                  │                              │     │
│  ▼                  ▼                              │     │
│  Q分支            空间重塑                          │     │
│  Linear(C→C)     x.view(B,C,H,W)                  │     │
│  │                  │                              │     │
│  ▼                  ▼                              │     │
│  reshape         Reduce (1×1 Conv)                 │     │
│  (B,heads,N,     C → C//4                          │     │
│   head_dim)         │                              │     │
│  │                  ▼                              │     │
│  │               DWT (Haar 2D)                     │     │
│  │               空间: H×W → H/2×W/2              │     │
│  │               通道: C//4 → C (4个子带拼接)       │     │
│  │                  │                              │     │
│  │                  ▼                              │     │
│  │               Filter (3×3 Conv + BN + ReLU)     │     │
│  │               增强/滤波小波系数                    │     │
│  │                  │                              │     │
│  │          ┌───────┼───────┐                      │     │
│  │          │               │                      │     │
│  │          ▼               ▼                      │     │
│  │      KV嵌入分支       IDWT分支                   │     │
│  │      kv_embed         IDWT (Haar 2D)            │     │
│  │      (Conv, stride=   H/2×W/2 → H×W            │     │
│  │       sr_ratio)       通道: C → C//4            │     │
│  │          │               │                      │     │
│  │          ▼               ▼                      │     │
│  │      LayerNorm        reshape                   │     │
│  │      Linear(C→2C)    (B, N, C//4)              │     │
│  │          │               │                      │     │
│  │          ▼               │                      │     │
│  │      K (B,heads,       │                       │     │
│  │         N',head_dim)    │                       │     │
│  │      V (B,heads,       │                       │     │
│  │         N',head_dim)    │                       │     │
│  │          │               │                      │     │
│  ▼──────────▼               │                      │     │
│                             │                      │     │
│  Q @ K^T × scale            │                      │     │
│      │                      │                      │     │
│      ▼                      │                      │     │
│  Softmax                    │                      │     │
│      │                      │                      │     │
│      ▼                      │                      │     │
│  Attn @ V                   │                      │     │
│      │                      │                      │     │
│      ▼                      │                      │     │
│  reshape (B, N, C)          │                      │     │
│      │                      │                      │     │
│      └──────┬───────────────┘                      │     │
│             │                                      │     │
│             ▼                                      │     │
│        Concat [attn_out, x_idwt]                   │     │
│        dim=-1 → (B, N, C + C//4)                   │     │
│             │                                      │     │
│             ▼                                      │     │
│        Linear(C + C//4 → C)                        │     │
│             │                                      │     │
└─────────────┼──────────────────────────────────────┘     │
              │                                            │
              ▼                                            │
        (+) Add ◄──────────────────────────────────────────┘
              │
              ├────────────────────────────────────────────┐ (residual)
              │                                            │
              ▼                                            │
        LayerNorm                                          │
              │                                            │
              ▼                                            │
        ┌──── PVT2FFN ────┐                                │
        │  Linear(C→hid)  │                                │
        │      │          │                                │
        │      ▼          │                                │
        │  DWConv 3×3     │                                │
        │      │          │                                │
        │      ▼          │                                │
        │   GELU          │                                │
        │      │          │                                │
        │      ▼          │                                │
        │  Linear(hid→C)  │                                │
        └──────┼──────────┘                                │
               │                                           │
               ▼                                           │
         (+) Add ◄─────────────────────────────────────────┘
               │
               ▼
         输出 x (B, N, C)
```

> **N' = (H/2 × W/2) / sr_ratio²**，即 K/V 的 token 数远小于 Q 的 token 数 N = H×W。

## 二、完整 WaveViT 网络结构流转图

```
输入图像 (B, 3, H, W)
│
▼
╔══════════════════════════════════════════════════════╗
║  Stage 1 — Stem                                     ║
║  Conv 7×7 s2 → BN → ReLU                            ║
║  Conv 3×3 s1 → BN → ReLU                            ║
║  Conv 3×3 s1 → BN → ReLU                            ║
║  Conv 3×3 s2 → LayerNorm                             ║
║  输出: (B, H/4 × W/4, C1)      C1=64                ║
╚══════════════════════╤═══════════════════════════════╝
                       │
                       ▼
            ┌─────────────────────┐
            │  WaveBlock × 3      │ ◄── WaveAttention + PVT2FFN
            │  heads=2, sr_ratio=4│     (小波注意力, 高分辨率)
            └──────────┬──────────┘
                       │
                       ▼
                  LayerNorm
                  reshape → (B, C1, H/4, W/4)
                       │
                       ▼
╔══════════════════════════════════════════════════════╗
║  Stage 2 — DownSample                               ║
║  Conv 3×3 s2 → LayerNorm                             ║
║  输出: (B, H/8 × W/8, C2)      C2=128               ║
╚══════════════════════╤═══════════════════════════════╝
                       │
                       ▼
            ┌─────────────────────┐
            │  WaveBlock × 4      │ ◄── WaveAttention + PVT2FFN
            │  heads=4, sr_ratio=2│     (小波注意力, 中分辨率)
            └──────────┬──────────┘
                       │
                       ▼
                  LayerNorm
                  reshape → (B, C2, H/8, W/8)
                       │
                       ▼
╔══════════════════════════════════════════════════════╗
║  Stage 3 — DownSample                               ║
║  Conv 3×3 s2 → LayerNorm                             ║
║  输出: (B, H/16 × W/16, C3)    C3=320               ║
╚══════════════════════╤═══════════════════════════════╝
                       │
                       ▼
            ┌─────────────────────┐
            │  StdBlock × 6       │ ◄── 标准Attention + PVT2FFN
            │  heads=10           │     (标准注意力, 低分辨率)
            └──────────┬──────────┘
                       │
                       ▼
                  LayerNorm
                  reshape → (B, C3, H/16, W/16)
                       │
                       ▼
╔══════════════════════════════════════════════════════╗
║  Stage 4 — DownSample                               ║
║  Conv 3×3 s2 → LayerNorm                             ║
║  输出: (B, H/32 × W/32, C4)    C4=448               ║
╚══════════════════════╤═══════════════════════════════╝
                       │
                       ▼
            ┌─────────────────────┐
            │  StdBlock × 3       │ ◄── 标准Attention + PVT2FFN
            │  heads=14           │     (标准注意力, 最低分辨率)
            └──────────┬──────────┘
                       │
                       ▼
╔══════════════════════════════════════════════════════╗
║  Post Network — ClassBlock                           ║
║                                                      ║
║  cls_token = mean(x, dim=1)     全局平均池化生成CLS    ║
║  x = cat([cls_token, x])                             ║
║      │                                               ║
║      ▼                                               ║
║  ┌─ ClassAttention ─────────────────────────────┐    ║
║  │  Q ← cls_token only    (1 token)             │    ║
║  │  K, V ← all tokens     (N+1 tokens)          │    ║
║  │  cls = softmax(Q @ K^T / √d) @ V             │    ║
║  └──────────────────────────────────────────────┘    ║
║      │                                               ║
║      ▼                                               ║
║  cls_token + Attention(cls_token)                    ║
║      │                                               ║
║      ▼                                               ║
║  cls_token + FFN(cls_token)                          ║
║      │                                               ║
║      ▼                                               ║
║  输出: [cls_token, patch_tokens]                      ║
╚══════════════════════╤═══════════════════════════════╝
                       │
                       ▼
                  LayerNorm
                       │
               ┌───────┴───────┐
               │               │
               ▼               ▼
          cls_token       patch_tokens
          (B, C4)         (B, N, C4)
               │               │
               ▼               ▼
          Head Linear     Aux Head Linear
          (C4 → classes)  (C4 → classes)
               │               │
               ▼               ▼
          x_cls            x_aux
               │               │
               └───────┬───────┘
                       │
                       ▼
              推理: x_cls + 0.5 × max(x_aux)
              训练: (x_cls, x_aux, bbox)
```

## 三、设计要点总结

```
┌──────────────────────────────────────────────────────────────┐
│                    WaveViT 设计哲学                           │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│  高分辨率 Stage (1,2)         低分辨率 Stage (3,4)            │
│  ┌──────────────────┐        ┌──────────────────┐           │
│  │  WaveAttention   │        │  Std Attention    │           │
│  │                  │        │                   │           │
│  │  • DWT 压缩 K/V  │        │  • 全量 Q/K/V     │           │
│  │  • IDWT 恢复细节  │        │  • token 数已少    │           │
│  │  • 复杂度 ≈ O(N²/4)│       │  • 复杂度可接受    │           │
│  └──────────────────┘        └──────────────────┘           │
│                                                              │
│  全局语义 (注意力路径)    +    局部细节 (IDWT 旁路)            │
│  ─────────────────────────────────────────────               │
│          Concat + Linear 融合 → 两全其美                      │
│                                                              │
└──────────────────────────────────────────────────────────────┘
```

> **wavevit_s 默认配置**: embed_dims=[64,128,320,448], depths=[3,4,6,3], heads=[2,4,10,14], sr_ratios=[4,2,1,1]
