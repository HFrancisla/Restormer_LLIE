# DualAttention 修改前后对比分析

## 1. 结构差异

| | 修改前（5 组独立投影） | 修改后（3 组共享投影） |
|---|---|---|
| **1×1 Conv** | `dim → dim*5` | `dim → dim*3` |
| **3×3 DWConv** | `dim*5 → dim*5` | `dim*3 → dim*3` |
| **空间 Q_s, K_s** | 独立权重，`Q_s ≠ K_s` | 共享 X_s，`Q_s = K_s` |
| **频率 Q_f, K_f** | 独立权重，`Q_f ≠ K_f` | 共享 X_f，`Q_f = K_f` |

---

## 2. 参数量变化

以单个 `DualAttention` 层、`dim=48` 为例：

- **1×1 Conv**: 48×(5×48) → 48×(3×48)，减少 **4,608** 参数
- **3×3 DWConv**: (5×48)×9 → (3×48)×9，减少 **864** 参数
- 每层共减少约 **5,472** 参数（减少 40% 投影参数）

---

## 3. 对网络的影响

### 3.1 空间注意力：对称 vs 非对称

**修改前** — `attn = Q_s @ K_s^T`：

- Q_s 和 K_s 是不同线性变换的输出
- 注意力矩阵**非对称**：通道 i 关注通道 j 的程度 ≠ 通道 j 关注通道 i 的程度
- 可以建模**有向关系**（如：亮度通道单向引导色度通道）

**修改后** — `attn = X_s @ X_s^T`：

- 注意力矩阵**对称且半正定**：`attn[i,j] == attn[j,i]`
- 退化为通道间的**自相关矩阵**，只能建模**互相关强度**，不能建模方向性
- 表达能力降低，但参数更少、更稳定

### 3.2 频率分支：能量谱 vs 交叉谱

**修改前** — `Q_f_fft * K_f_fft`：

- 频域中两个不同信号的交叉谱，结果是**复数**，IFFT 后可正可负
- 调制信号能同时**增强和抑制**空间特征的不同分量

**修改后** — `X_f_fft * X_f_fft = |X_f_fft|²`（功率谱）：

- 结果是**实数且恒非负**，IFFT 后也倾向于非负
- 调制信号只能**增强**，难以**抑制**
- 依赖后续 LayerNorm 的 re-center 来恢复正负极性

### 3.3 实际训练影响预期

| 维度 | 影响 |
|---|---|
| **收敛速度** | 参数减少 → 可能更快收敛，但容量下降可能限制极限性能 |
| **过拟合风险** | 参数更少 → 小数据集上可能更好泛化 |
| **低光增强** | 频率分支恒非负可能导致暗区细节恢复能力下降，因为无法抑制噪声频率分量 |
| **训练稳定性** | 对称注意力 + 非负调制 → 梯度更平滑，训练更稳定 |

---

## 4. 总结

修改后严格对齐了图示架构，参数更少、训练更稳定，但**表达能力下降**——空间分支丢失了方向性建模，频率分支丢失了抑制能力。如果低光增强任务需要精细的噪声抑制，修改前的独立投影版本可能效果更好。
